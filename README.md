# GIFTRANDOM.RU
Уникальный сервис для тех, кто не знает что подарить своему другу, подруге, любимому человек или родным. Веб-сервис GiftRandom поможет Вам определиться с подарком. Просто нажмите на кнопку и узнайте что подарить.

# v.1.0
- Имеется возможность записи новых объектов в БД.
- Имеется возможность управления объектами БД (удаление, изменение).
- При выполнения поиска объекта происходит запрос к БД (каждый раз при нажатии на кнопку).
- Доступ в панель администратора ограничена правами доступа (логин - пароль).
- Сессионая авторизация в панель администратора.
